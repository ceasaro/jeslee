=============
Product Taxes
=============

This section describes the management interface for product taxes.

Site Actions
============

Add Tax
    Adds a new product tax.

Delete Tax
    Deletes the current product tax.

Data
====

Rate
    The tax rate in percent.

Description
    A short description for internal uses only.
